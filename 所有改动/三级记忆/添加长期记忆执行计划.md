# 添加长期记忆（语义检索层）执行计划

## 1. 项目概述

### 1.1 项目背景
智扫通机器人智能客服系统已完成短期记忆（会话层）和中期记忆（文件层）的实现。为应对长期运行中日志规模膨胀和语义检索需求，现启动第三层记忆系统——语义检索层的开发工作。

### 1.2 项目目标
构建基于向量检索的语义记忆系统，实现对海量历史日志的智能检索，形成完整的三层记忆架构：
- 第一层：会话记忆（MemoryManager）- 实时上下文
- 第二层：文件记忆（FileMemory）- 结构化摘要  
- 第三层：语义检索（SemanticMemory）- 向量化检索

### 1.3 项目范围
**包含**：
1. SemanticMemory 类设计与实现
2. 基于 ChromaDB 的向量存储后端
3. 日志文件自动索引和增量更新
4. 语义检索 API 和结果排序
5. 与现有 ReactAgent 集成
6. 配置系统和监控指标

**不包含**：
1. 更换 embedding 模型（复用现有 sentence-transformers）
2. 修改现有 MemoryManager 和 FileMemory 核心逻辑
3. 用户界面开发（纯后端服务）
4. 多模态记忆支持（图片、表格等）

### 2.2 核心类设计
```python
# agent/semantic_memory.py
class SemanticMemory:
    """语义检索层管理器"""
    
    def __init__(self, config: Dict):
        self.config = config
        self.vector_store = None  # ChromaDB 实例
        self.embedding_model = None  # sentence-transformers 模型
        self._initialize_components()
    
    def _initialize_components(self):
        """初始化向量存储和 embedding 模型"""
        
    def index_log_file(self, log_path: Path) -> int:
        """索引单个日志文件，返回索引的块数"""
        
    def index_logs_directory(self, logs_dir: Path, force_reindex: bool = False) -> Dict:
        """批量索引日志目录"""
        
    def search(self, query: str, top_k: int = 5, 
               min_score: float = 0.0) -> List[MemoryResult]:
        """语义检索相关记忆"""
        
    def search_with_context(self, query: str, context: Dict = None) -> str:
        """检索并格式化为上下文文本"""
        
    def get_stats(self) -> Dict[str, Any]:
        """获取索引统计信息"""
        
    def cleanup_old_indices(self, max_age_days: int = 90):
        """清理过期索引（基于日志日期）"""
```

### 2.3 数据结构
```python
@dataclass
class MemoryResult:
    """检索结果数据结构"""
    text: str                    # 原始文本内容
    score: float                 # 相似度分数 (0.0-1.0)
    metadata: Dict[str, Any]     # 元数据
    source_file: str             # 来源文件
    log_date: date               # 日志日期
    chunk_index: int             # 块索引
    
    def to_context_format(self) -> str:
        """转换为上下文注入格式"""
        return f"[{self.log_date}] {self.text[:200]}..."
```

   from langchain_chroma import Chroma
   from sentence_transformers import SentenceTransformer
   
   class SemanticMemory:
       def _init_vector_store(self):
           # 初始化 ChromaDB，使用独立集合
           self.embedding_model = SentenceTransformer(
               self.config["embedding"]["model_name"],
               cache_folder=self.config["embedding"]["cache_dir"]
           )
           
           # 创建自定义 embedding 函数
           def custom_embedding_function(texts):
               embeddings = self.embedding_model.encode(texts)
               return embeddings.tolist()
           
           self.vector_store = Chroma(
               collection_name=self.config["vector_db"]["collection_name"],
               embedding_function=custom_embedding_function,
               persist_directory=self.config["vector_db"]["path"]
           )
