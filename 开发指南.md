## 故障排除

### 常见问题

#### 1. 依赖缺失错误
```
ImportError: 缺少必要的依赖库：langchain-chroma 或 sentence-transformers
```
**解决方案**:
```bash
pip install langchain-chroma sentence-transformers
```

#### 2. 向量库连接失败
**可能原因**:
- 存储路径不可写
- ChromaDB版本不兼容
- 磁盘空间不足

**解决方案**:
- 检查路径权限
- 确认ChromaDB版本 >= 0.4.0
- 清理磁盘空间

#### 3. 内存使用过高
**可能原因**:
- 同时索引过多文件
- embedding模型过大

**解决方案**:
- 减少`max_workers`数量
- 使用更小的embedding模型
- 增加`max_chunks_per_file`限制

### 故障排除

#### 常见问题
1. **语义记忆未启用**
   - 检查`agent.yml`中`semantic_memory.enabled`是否为`true`
   - 确认`SemanticMemory`类导入路径正确

2. **向量数据库连接失败**
   - 检查`vector_db.path`目录是否存在且可写
   - 确认ChromaDB版本兼容性（>=0.4.0）

3. **自动索引未执行**
   - 检查`indexing.auto_index`配置
   - 确认`./memory/logs`目录存在且包含日志文件
   - 查看控制台日志是否有错误信息

4. **记忆查询返回空结果**
   - 确认日志文件已正确索引（检查索引统计）
   - 尝试降低`min_similarity_score`阈值
   - 验证查询语句是否包含相关关键词
